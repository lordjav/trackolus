Trackolus

Versión 1.0

Trackolus es un sistema de control de almacenamiento tipo Sistema de Gestión de Almacenes (WMS por sus siglas en inglés) diseñado para controlar, supervisar y verificar el almacenamiento de una o varias bodegas de manera centralizada. Su facilidad de uso y simplicidad en la organización de la información lo hacen una herramienta apropiada para pequeñas y medianas empresas.

Tecnologías usadas


Frontend:

La interfaz de usuario se basa principalmente en HTML, CSS y JavaScript. Para la comunicación con el servidor y procesamiento de las respuestas se usan herramientas de HTMX. Para el módulo de calendario se utilizó fullCalendar y algunos componentes visuales utilizan Bootstrap.

  • HTML5.
  • CSS 3.
  • JavaScript.
  • Bootstrap 5.3.3.
  • HTMX 1.9.7.
  • FullCalendar 6.1.15.

Backend:

La lógica de la aplicación se construyó en Python con Flask como framework y usando Jinja2 para el renderizado de las plantillas. Las gráficas fueron construidas con SQLAlchemy para las consultas, Pandas para la estructuración y Plotly para el renderizado. La traducción de la app se basó en un 50% en Babel y el resto en soluciones propias. PDFKIT se utilizó en los módulos de generación de documentos pdf.

  • Python 3.12.3.
  • Flask 3.0.3.
  • Babel. 2.16.0.
  • Jinja2 3.1.4.
  • Pandas.
  • Plotly.
  • PDFKIT.

Base de datos:

SQLite3 usando mediante el módulo SQL de CS50 como ORM, otra parte del manejo de la información utilizó SQLAlchemy para la visualización de gráficas y para el registro de errores.

Instalación

Puesto que se trata de una aplicación web solo se requiere un navegador y no necesita de instalación, basta con acceder mediante la URL para su uso. Se recomienda tener la última versión del navegador preferido del usuario para evitar problemas de incompatibilidad con las tecnologías usadas. Se desaconseja el uso de Internet Explorer 13 y versiones anteriores.

Uso

Al iniciar sesión se observan 6 secciones principales accesibles mediante la barra lateral izquierda:

Tablero: una recopilación de gráficos para análisis visual del almacenamiento en la(s) bodegas(s). Con información sobre ingresos y egresos en el tiempo, productos con menor stock, evolución en los últimos días así como comparativos con la última semana, mes y trimestre, etc.

Reportes: acceso a reportes según varios tipos de datos como productos, clientes, proveedores, actividad de los usuarios, entradas y salidas de mercancía, entre otros. Se ofrece la generación de los informes en diferentes formatos: documentos pdf, hojas de cálculo y archivos de datos tipo CSV.

Inventario: Listado de todos los productos almacenados con su respectiva información de códigos SKU, existencias, distribución por almacén, costo, imágenes, etc. También en esta sección se accede a la adición de nuevos productos y la transferencia de mercancía entre almacenes.

Para agregar nuevos productos acceda al menú desplegable mediante el botón en la esquina inferior derecha, ingrese la información del producto, seleccione una imagen de referencia y guarde la información.

Para transferir artículos entre almacenes acceda mediante el mismo menú, defina almacén de origen y destino, seleccione los artículos y guarde la orden de transferencia.

Orden de compra: con su interfaz sencilla se agregan órdenes de salida de mercancía en pocos pasos. También con opción para la generación de cotizaciones. Para agregar una orden ingrese los datos del cliente (nombre, teléfono, identificación y correo). En caso de existir el cliente en la base de datos se cargará su información en los campos automáticamente. De lo contrario, se guardará la información en la base de datos por defecto. Seleccione luego la bodega desde la cuál saldrá la mercancía, los productos a incluir en la orden y su cantidad. Puede guardar la orden de compra o generar una cotización en pdf.

Entradas: para consultar órdenes de ingreso de mercancía y su contenido además de información adicional, informes de recepción de productos y el registro de nuevos ingresos.

Salidas: también en formato de listado para visualización rápida de egreso de mercancía.

Otras funciones:

  • Calendario: para visualización de órdenes en órden cronológico con opción de vista por agenda, día, semana y mes. Acceso mediante el menú en la esquina superior derecha.
  • Notificaciones: registro en tiempo real de los eventos más importantes como entradas y salidas de mercancía, transferencia entre almacenes, adición y edición de nuevos productos, entre otros. Acceso mediante el menú en la esquina superior derecha.
  • Lenguaje: con soporte para inglés y español. Menú de selección en la esquina superior derecha.
